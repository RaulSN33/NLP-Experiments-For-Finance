# NLP-Experiments-For-Finance

This repo contains the code and report for the analysis of earnings call sentiment and its ability to predict subsequent stock market reactions, measured by cumulative abnormal returns
from the Carhart 4 factor model. The objective is to build multiple sentiment measures from earnings call transcripts using both traditional Bag-of-Words (BoW) approaches and modern large language models (LLMs), compare how these measures relate to one another, and evaluate which one best captures the informational content that moves stock prices after an earnings call.
